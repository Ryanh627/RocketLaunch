Information about this directory:

1) This is meant to contain a python virtual environment, which isolates
   dependencies used by Python such as external libraries installed with pip

2) To activate a virtual environment, use the 'source' command pointed at
   the activate.py file located in VirtualEnv/env/bin/

   Example: source env/bin/activate.py

3) To exit a virtual environment, simply use the 'deactivate' command
