Information about this directory:

1) /Source leads to the web application source code and the python virtual
   environment directory

2) /Installation contains files that help users or developers to fulfill
   system requirements for running the web application

3) /Tasks contains documents listing actions the developers have taken on
   particular meeting days

4) /Stories contains documents listing use cases that the user wants to perform

5) /Dependencies contains a text document listing install commands used and an
   Apache configuration file used by installers

6) /Spikes contains experiments undertaken by developers to gain experience with
   particular tasks before implementing them as part of the main project

7) /Standards contains documents detailing standards the developers are to abide     by

8) /Design contains items relating to the design process of the application
